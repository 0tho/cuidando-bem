Module: Stage

Stage

The module stage is responsible to change and control the "screen" presented to the user. For the module work it is needed to register "views" and "controllers" before calling the method start

A view is a html file, and a controller is a js file that is responsible to make its view work. To bind all events to the view, as for example button clicks. Because this project uses require.js every call to a new "screen" will asynchronously load the new view and controller
